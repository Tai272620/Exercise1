What is HTML?

HTML stands for HyperText Markup Language. It's markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links, heading, images, videp, etc. Web browsers understand HTML and render HTML code as website.

In HTML, each element is made up of 3 parts:

  1. THE opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDM Web Docs.

Why should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix
  • To impress friends
  • To have fun
